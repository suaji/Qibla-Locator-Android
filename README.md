# Qibla Locator Android App 🕋

A lightweight and precise Android application designed to help users find the Qibla direction using device sensors. <br>
Built with a focus on accuracy, performance, and a minimalist user interface. <br>

## ✨ Key Features
* **Real-time Compass:** High-precision orientation tracking using Magnetometer and Accelerometer sensors. <br>
* **Manual Offset Adjustment:** Includes intuitive `+1°` and `-1°` buttons for fine-tuning sensor calibration or temperature offsets. <br>

## 🛠️ Technical Specifications
This app is built following modern Android development standards: <br>
* **Programming Language:** Kotlin <br>
* **Minimum Version:** Android 6.0 Marshmallow <br>

## 💻 Notes
This application requires access to the device's compass sensors for optimal performance. <br>
Accuracy may vary based on environmental interference and device hardware capabilities. <br>
